## 1. Резюме для принятия решений

### 1.1. Что такое Truffles

**Truffles** — это суверенная AI-платформа для автоматизации бизнес-коммуникаций, созданная специально для рынка Казахстана и Центральной Азии.

**Ключевая формула:**
```
Truffles = Умный бот-автоответчик + Kaspi Pay проверка + Локализация данных
```

> **ВАЖНО О ПОЗИЦИОНИРОВАНИИ:**
> - Для МСБ: продаём "автоответчик" и "проверку оплат", НЕ "AI"
> - Для Enterprise: продаём "compliance" и "контроль", AI как технология
> - Слово "AI" — для инвесторов и технической аудитории

### 1.2. Почему именно сейчас

### 1.3. Уникальность позиционирования

| Конкурент | Их слабость | Наше преимущество |
|-----------|-------------|-------------------|
| Umnico, TextBack | Данные в облаке (РФ/EU), нет AI | Локальное хранение + настоящий AI |
| Bitrix24 CoPilot | Закрытая система, нет Kaspi | Открытая логика + глубокая интеграция |
| Заказная разработка | Дорого, долго, риск «bus factor» | Продукт + поддержка + модули |
| ChatGPT/Claude напрямую | Нет интеграций, нет истории | Полная платформа с CRM и платежами |

## 2. Целевые сегменты и их боли

### 2.1. Сегмент A: Микро и малый бизнес (SaaS Starter)

**Портрет:**
- Салоны красоты, автосервисы, небольшие магазины
- 1-5 сотрудников
- Оборот: 3-20 млн ₸/год
- Владелец = менеджер = бухгалтер

**Главные боли:**
```
1. "Не успеваю отвечать в WhatsApp — теряю клиентов"
2. "Одни и те же вопросы каждый день"
3. "Клиенты присылают фейковые скриншоты Kaspi"
4. "Не могу позволить себе менеджера на телефон"
```

**Что они хотят:**
- Простое решение «из коробки»
- Цена < зарплаты сотрудника
- Работает сразу, без IT-специалиста

**Willingness to Pay:** 30,000 – 70,000 ₸/месяц

### 2.2. Сегмент B: Средний бизнес (SaaS Pro)

**Портрет:**
- Сети салонов, клиники, агентства недвижимости
- 10-50 сотрудников
- Оборот: 50-500 млн ₸/год
- Есть отдельный маркетолог или руководитель продаж

**Главные боли:**
```
1. "Менеджеры по-разному отвечают — нет стандарта"
2. "CRM не связана с мессенджерами"
3. "Не вижу, кто из клиентов оплатил"
4. "Хочу рассылки, но боюсь блокировки"
```

**Что они хотят:**
- Интеграция с их CRM (AmoCRM, Bitrix)
- Автоматическая проверка оплаты Kaspi
- Аналитика и отчёты
- Несколько каналов (WhatsApp + Instagram)

**Willingness to Pay:** 100,000 – 200,000 ₸/месяц

### 2.3. Сегмент C: Крупный бизнес (Enterprise / On-Premise)

**Портрет:**
- Банки, страховые, застройщики, мед. сети
- 100+ сотрудников
- Оборот: 1+ млрд ₸/год
- Есть IT-отдел и служба безопасности

**Главные боли:**
```
1. "Не можем хранить данные клиентов в чужом облаке — закон"
2. "Нужен audit trail для compliance"
3. "Хотим интеграцию с 1С и внутренними системами"
4. "Боимся vendor lock-in"
```

**Что они хотят:**
- Установка на свои серверы
- Полный контроль над данными
- Исходный код / открытая архитектура
- SLA и enterprise support

**Willingness to Pay:** 5-15 млн ₸ единоразово + 1-2 млн ₸/год поддержка

### 3.4. Детализация тарифа Enterprise

**Модель:** Perpetual License (вечная лицензия) + Annual Maintenance

**Структура сделки:**

| Компонент | Сумма | Описание |

| **License Fee** | 2,000,000 – 6,500,000 ₸ | Право на использование навсегда |
| **Annual Maintenance** | 18-20% от License | Обновления, поддержка, совместимость |

**Пример: Медицинский центр (средний)**
```
Внедрение:      750,000 ₸
Лицензия:     2,100,000 ₸
────────────────────────────
Год 1:        2,850,000 ₸

AMF (20%):      420,000 ₸/год (со 2-го года)
```

**Пример: Банк / Страховая**
```
Внедрение:    2,500,000 ₸
Лицензия:     6,500,000 ₸
────────────────────────────
Год 1:        9,000,000 ₸

AMF (18%):    1,170,000 ₸/год (со 2-го года)
```

**Что включено в Enterprise:**

| Функция | Описание |
|---------|----------|
| Развёртывание On-Premise | На серверах клиента |
| Неограниченные пользователи | Без лимита |
| Неограниченные сообщения | BYOK (клиент платит за API сам) |
| Все модули | Kaspi, CRM, 1C, рассылки |
| Локальный LLM (опция) | Ollama для чувствительных данных |
| SSO интеграция | Active Directory, Google Workspace |
| Исходный код воркфлоу | Полная прозрачность |
| SLA | 99.5% uptime, 4ч response |
| Выделенный менеджер | Персональная поддержка |

### 3.5. Managed On-Premise (Подписка + Сервер клиента)

**Гибридная модель для компаний, которым нужен контроль над данными, но без капитальных затрат.**

```
┌─────────────────────────────────────────────────────────────────┐
│                   MANAGED ON-PREMISE                            │
│         "Ваш сервер — наше управление и поддержка"             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  КЛИЕНТ ПРЕДОСТАВЛЯЕТ:                                         │
│  ├── VPS/сервер (свой или арендованный)                        │
│  ├── Минимум: 4 vCPU, 8 GB RAM, 100 GB SSD                     │
│  └── SSH-доступ для развёртывания                              │
│                                                                 │
│  МЫ ПРЕДОСТАВЛЯЕМ:                                             │
│  ├── Развёртывание и настройку платформы                       │
│  ├── Регулярные обновления и патчи                             │
│  ├── Мониторинг и алерты                                       │
│  ├── Техническую поддержку (SLA)                               │
│  └── Резервное копирование (опционально)                       │
│                                                                 │
│  ПРЕИМУЩЕСТВА:                                                  │
│  ├── Данные физически у клиента = полный комплаенс             │
│  ├── Нет капитальных затрат на лицензию                        │
│  ├── Предсказуемые ежемесячные расходы                         │
│  └── Возможность выкупа лицензии позже                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**Структура ценообразования:**

| Компонент | Сумма | Описание |
|-----------|-------|----------|
| **Setup Fee** | 200,000 – 500,000 тг | Разовая настройка и развёртывание |
| **Ежемесячная подписка** | 100,000 – 200,000 тг | Поддержка + обновления + мониторинг |
| **Инфраструктура** | Оплачивает клиент | VPS ~15,000-50,000 тг/мес |

**Пример: Логистическая компания**
```
Setup Fee:              300,000 тг (разово)
Подписка:               150,000 тг/мес
VPS (клиент сам):        30,000 тг/мес
────────────────────────────────────────
Год 1 (всего):        2,460,000 тг
Год 2+ (всего):       2,160,000 тг/год
```

**Сравнение с другими моделями:**

| Параметр | SaaS Pro | Managed On-Premise | Enterprise (Perpetual) |
|----------|----------|-------------------|------------------------|
| Начальные затраты | 0 | 200-500k тг | 2.8-9M тг |
| Ежемесячно | 150k тг | 100-200k тг | AMF: 35-100k тг |
| Где данные | Наш сервер (КЗ) | Сервер клиента | Сервер клиента |
| Кто управляет | Мы | Мы | Клиент (мы поддерживаем) |
| Масштабирование | Ограничено тарифом | Гибкое | Безлимит |
| Выход из контракта | Легко | Средне | Сохраняет лицензию |

**Идеально для:**
- Компаний с требованиями к локализации данных
- Бизнеса, который хочет контроль, но не готов к капзатратам
- Тех, кто планирует масштабироваться и выкупить лицензию позже
