## ЧТО ЭТО

**Elevator pitch:**
> Truffles — управляемый LLM‑консультант для WhatsApp. Клиент написал в 11 вечера — получил ответ и следующий шаг к записи.

**Для кого:**
Малый и средний бизнес Казахстана с 20+ сообщениями в день в WhatsApp.

**Ценность:**
- Для клиента: ответ в любое время, не ждать до утра
- Для бизнеса: менеджер не нужен 24/7, бот закрывает 80% вопросов

**Позиционирование:**
- Для МСБ: продаём "консультант + запись", НЕ "AI"
- Слово "AI" — для инвесторов и технической аудитории

## UPSELL PATH

```
STARTER → PRO когда:
- Заканчиваются 1,000 сообщений
- Нужен второй номер (филиал)
- Хочет видеть диалоги (dashboard)
- Хочет чтобы бот учился
- Нужна проверка Kaspi Pay

PRO → ENTERPRISE когда:
- Сеть/франшиза (много точек)
- Нужен свой бренд (white-label)
- Нужна интеграция с внутренними системами
- Требуется SLA/гарантии
```
