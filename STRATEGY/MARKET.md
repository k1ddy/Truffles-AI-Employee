## ЦЕЛЕВЫЕ НИШИ

### Tier 1: Идеальные клиенты (начинать с них)

#### Салоны красоты / Барбершопы
**Почему:**
- Уже живут в WhatsApp
- Типовые вопросы: цены, запись, адрес
- Владелец = менеджер (боль чувствует лично)
- Готовы платить 50К (< зарплаты уборщицы)

**Типичные вопросы:**
- Сколько стоит стрижка/маникюр?
- Есть свободное время на завтра?
- Можно к Айгуль записаться?

**Где искать:** Instagram по геолокации, 2GIS

---

#### Магазины Instagram / Шоурумы
**Почему:**
- WhatsApp = основной канал продаж
- Много одинаковых вопросов
- Боль с Kaspi скриншотами

**Типичные вопросы:**
- Есть в наличии?
- Какие размеры?
- Сколько доставка?
- Оплата Kaspi?

**Где искать:** #шоурумалматы, #одеждаалматы

---

#### Кондитерские / Торты на заказ
**Почему:**
- Эмоциональная покупка (срочно на ДР)
- Простой FAQ: цена/кг, начинки, доставка
- Владелец = кондитер, руки заняты

**Типичные вопросы:**
- Сколько стоит торт на 2 кг?
- Какие начинки есть?
- Можно срочно на сегодня?

---

### Tier 2: Хорошие клиенты (после 3-5 из Tier 1)

#### Автосервисы
- Высокий средний чек
- Постоянные клиенты
- Сложнее объяснить ценность (владельцы 40+)

#### Доставка еды
- Много вопросов
- Конкуренция с Glovo/Wolt
- Низкая маржа

---

### Tier 3: Позже

#### Медицина / Клиники
- Высокие требования
- Сложная специфика

#### Enterprise
- Длинный цикл продаж
- Нужен SLA

## ICP (Ideal Customer Profile)

### Идеальный клиент Truffles:

| Параметр | Значение |
|----------|----------|
| Ниша | Салон, магазин, кондитерская |
| Размер | 1-10 сотрудников |
| Оборот | 3-50 млн ₸/год |
| Сообщений/день | 20-100 |
| Канал | WhatsApp основной |
| Боль | "Не успеваю отвечать" |
| Budget | 50-150К ₸/мес |

### Красные флаги (НЕ брать):
- "У нас нет WhatsApp, только звонки"
- "Мы и так справляемся"
- < 10 сообщений в день
- Не принимают Kaspi

## АНТИПАТТЕРНЫ (что НЕ делать)

| Антипаттерн | Почему плохо | Что делать |
|-------------|--------------|------------|
| Один бот на всё | Alibaba доказал: специализация лучше | Разделять по задачам |
| Мерить Deflection Rate | Скрывает фрустрацию | Quality Deflection |
| Надеяться на LLM | 27% галлюцинаций | RAG + confidence |
| "Нужно больше данных" | 25 примеров = 94% | Начать с малого |
| Fine-tuning решит | Не снижает галлюцинации | RAG + валидация |
| Бот заменит продажника | 75% не справляются | Бот усиливает человека |

## ПАТТЕРНЫ УСПЕХА

| Паттерн | Как применить |
|---------|---------------|
| **RAG Only** | Бот говорит ТОЛЬКО из базы |
| **Confidence Threshold** | <70% → эскалация |
| **Whitelist для критичного** | Цены, возвраты — фиксированные ответы |
| **Эскалация за 1 сообщение** | Клиент всегда может выйти |
| **Прозрачность** | "Я не уверен, передам менеджеру" |
