# Workflow

Здесь хранится n8n workflow.

## Файлы

- `main.json` — основной workflow (будет создан)

## Принцип

ОДИН workflow. Не плодить сложность. Всё в одном месте = проще дебажить.

## Что должен делать workflow

```
1. Принять сообщение (Webhook)
2. Загрузить историю диалога (PostgreSQL)
3. Найти релевантный контекст (Qdrant RAG)
4. Сгенерировать ответ (GPT-4o)
5. Сохранить в историю (PostgreSQL)
6. Отправить ответ (WhatsApp)
7. Обработать ошибки (Fallback к менеджеру)
```
