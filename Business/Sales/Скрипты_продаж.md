# Скрипты продаж Truffles

---

## 1. Первый контакт (холодный)

### WhatsApp / Telegram знакомому

```
Привет! Как дела?

Слушай, делаю проект — бот для бизнеса, который отвечает 
в WhatsApp и проверяет оплату Kaspi автоматом.

Хочу протестировать на реальном бизнесе. Бесплатно месяц.
Взамен — честный feedback.

Есть 15 минут на созвон? Покажу как работает.
```

### Если спросит "А что умеет?"

```
Если коротко:
- Отвечает на частые вопросы (цены, запись, адрес)
- Если сложный вопрос — сразу тебе в Telegram
- Проверяет оплату Kaspi — никаких фейковых скринов

По сути замена менеджера на первой линии.
Ты подключаешься только когда реально нужен.
```

### Если спросит "Сколько стоит?"

```
50,000 в месяц. Но первый месяц бесплатно — тестируем вместе.
Если не понравится — просто отключаем, без проблем.
```

---

## 2. На встрече / созвоне

### Открытие (2 мин)

```
Спасибо что нашёл время.

Давай так: я сначала спрошу про твой бизнес — чтобы понять 
подходит ли вообще. Потом покажу как работает. И решим — 
имеет смысл тестировать или нет.

Ок?
```

### Вопросы про боль (5 мин)

```
1. Сколько примерно сообщений в день получаете в WhatsApp?

2. Какие вопросы чаще всего задают? 
   (записывай! это ключ к настройке)

3. Сколько времени в день уходит на ответы?

4. Бывает что клиенты уходят потому что долго отвечали?

5. Как сейчас проверяете оплату Kaspi? 
   Присылают скриншоты? Бывали фейковые?
```

### Демо (5 мин)

```
Смотри, покажу на примере.

[Открываешь WhatsApp, пишешь боту]

Вот, написал "Сколько стоит стрижка?"

[Бот отвечает]

Видишь — ответил за 3 секунды. Ночью, в выходные — без разницы.

А теперь напишу что-то сложное...

[Пишешь "Хочу поговорить с человеком"]

[Показываешь уведомление в Telegram]

Вот — мне сразу пришло. Я вижу контекст, могу ответить.
```

### Предложение (2 мин)

```
Вот что предлагаю:

1. Ты скидываешь прайс и частые вопросы — 15 минут
2. Я настраиваю бота под твой бизнес — день-два
3. Тестируем неделю
4. Если норм — работаем дальше

Первый месяц бесплатно. Потом 50,000 в месяц.
Если не понравится — просто отключаем.

Что скажешь?
```

## 3. Обработка возражений

### "Дорого"

```
Понимаю. Давай посчитаем:

Менеджер на телефон — минимум 150,000 в месяц.
Плюс налоги, отпуска, больничные.
И работает только в рабочее время.

Бот — 50,000. Работает 24/7. Не болеет. Не уходит.

Но я понимаю что для теста это может быть много.
Поэтому первый месяц — бесплатно.
Попробуешь — решишь сам.
```

### "У нас уже есть бот"

```
Отлично! А он умеет:
- Отвечать на произвольные вопросы? (не только кнопки)
- Проверять оплату Kaspi автоматом?
- Передавать сложные вопросы тебе?

Если нет — можем сравнить. Бесплатно на месяц.
```

### "Боты раздражают клиентов"

```
Согласен, плохие боты раздражают.

Но этот другой — он реально понимает вопрос и отвечает по делу.
А если не знает ответ — сразу передаёт человеку.

Клиент даже не поймёт что это бот, пока ты не скажешь.

Давай просто попробуем? На твоих реальных клиентах увидим.
```

### "Нужно подумать"

```
Конечно. Что именно смущает? 
Может я могу ответить на вопросы сейчас.

[Если реально нужно время:]

Ок, давай так — я напишу через пару дней.
Если будут вопросы — пиши в любое время.
```

### "А если бот ошибётся?"

```
Будет ошибаться, особенно первые дни.

Но:
1. Все сообщения я вижу — если что-то не так, сразу поправлю
2. Если бот не уверен — передаёт тебе
3. Каждую неделю улучшаем на основе реальных диалогов

Первые 2 недели — я мониторю плотно.
```

### "У меня специфический бизнес"

```
Расскажи подробнее? Какие вопросы задают?

[Слушаешь]

Понял. Да, это можно настроить. 
Просто загружаем твою информацию — и бот её знает.

Давай попробуем — бесплатно же. Увидим на практике.
```

## 4. Закрытие сделки

### Если готов

```
Отлично. Тогда так:

1. Заполни короткий бриф — я скину в WhatsApp
2. Пришли прайс и частые вопросы
3. Я настрою и напишу когда готово

Займёт день-два. Ок?
```

### Если сомневается

```
Слушай, понимаю что решение не быстрое.

Давай так — я пришлю тебе короткое описание, 
посмотришь спокойно. И если будут вопросы — напиши.

Но вот что скажу: чем раньше попробуешь — 
тем раньше перестанешь отвечать на одинаковые вопросы.

Первый месяц бесплатно. Риска никакого.
```

## 5. Follow-up

### Через 2 дня (если не ответил)

```
Привет! Как там — посмотрел про бота?
Если есть вопросы — пиши, отвечу.
```

### Через неделю (если всё ещё думает)

```
Привет! Проверяю — ещё актуально про бота?
Если сейчас не до этого — ок, напишу позже.
```

### После успешного теста (в конце месяца)

```
Привет! Месяц пролетел.

Как тебе бот? Помогает?

Если всё ок — продолжаем за 50,000/мес.
Если что-то не так — скажи, поправим.
```

## 6. Что НЕ говорить

❌ "Это AI на GPT-4" — им пофиг на технологии  
❌ "Интегрируется с любой CRM" — если не уверен  
❌ "Заменит всех менеджеров" — это пугает  
❌ "У нас Enterprise решение" — это отталкивает МСБ  
❌ "Я стартап" — это вызывает недоверие  

✅ "Умный бот" — понятно  
✅ "Работает в WhatsApp" — конкретно  
✅ "Проверяет оплату Kaspi" — ценность  
✅ "Первый месяц бесплатно" — снимает риск
